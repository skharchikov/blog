A Rust application that generates beautiful GitHub statistics and visualizes them as SVG images for your profile README.

## Features

- **Automated Statistics**: Fetches GitHub data using GraphQL API
- **SVG Generation**: Creates customizable overview and language statistics
- **Contribution Heatmap**: Animated visualization of contribution history (WIP)
- **GitHub Actions Integration**: Automated workflow for continuous updates

## Tech Stack

- **Rust**: High-performance systems programming
- **GraphQL Client**: Efficient GitHub API queries
- **Tokio**: Async runtime for concurrent operations
- **Serde**: JSON serialization and configuration management
- **Tracing**: Structured logging with Bunyan formatter

## Use Cases

Perfect for developers who want to showcase their GitHub activity on their profile with automatically updated, visually appealing statistics cards.
