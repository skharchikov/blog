A modern, fast personal blog built with Leptos framework and Rust, compiled to WebAssembly.

## Tech Stack

- **Leptos 0.6**: Reactive web framework for Rust
- **Rust**: Systems programming language
- **WebAssembly**: Compile target for web browsers
- **Trunk**: Build tool for Rust WASM applications
- **Pulldown-cmark**: Markdown parser

## Build Process

Projects are loaded from markdown files at build time, ensuring fast load times and no runtime overhead.
